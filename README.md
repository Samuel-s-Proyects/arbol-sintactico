# Arbol Sintactico — Visualizador Educativo

Herramienta interactiva para visualizar arboles sintacticos, desarrollada con Next.js para la clase de Compiladores.

## Instalacion y ejecucion

```bash
npm install
npm run dev
```

La aplicacion estara disponible en `http://localhost:3000`.

## Scripts disponibles

| Comando          | Descripcion                                    |
|------------------|------------------------------------------------|
| `npm run dev`    | Servidor de desarrollo con hot-reload           |
| `npm run build`  | Compilacion de produccion                       |
| `npm run start`  | Servidor de produccion (requiere build previo)  |
| `npm run lint`   | Analisis estatico con ESLint                    |

## Tecnologias

- **Next.js 16** — App Router, Server y Client Components
- **React 19** — con hooks modernos
- **TypeScript** — tipado estricto en toda la aplicacion
- **Tailwind CSS v4** — utilidades de estilos
- **Font Awesome Free** — iconografia
